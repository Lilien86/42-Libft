# lib42

Le projet `libft` de l'école 42 consiste en la création d'une bibliothèque de fonctions de base, souvent présentes dans les bibliothèques standard du langage C, ainsi que quelques fonctions spécifiques à l'environnement pédagogique de 42.


## Fonctions

- `isalpha`
- `isdigit`
- `isalnum`
- `isascii`
- `isprint`
- `strlen`
- `memset`
- `bzero`
- `memcpy`
- `memmove`
- `strlcpy`
- `strlcat`
- `toupper`
- `tolower`
- `strchr`
- `strrchr`
- `strncmp`
- `memchr`
- `memcmp`
- `strnstr`
- `atoi`

- `calloc`
- `strdup`

- `ft_substr`
- `ft_strjoin`
- `ft_strtrim`
- `ft_split`
- `ft_itoa`
- `ft_strmapi`
- `ft_striteri`
- `ft_putchar_fd`
- `ft_putstr_fd`
- `ft_putendl_fd`
- `ft_putnbr_fd`

## Fonctions Bonus
